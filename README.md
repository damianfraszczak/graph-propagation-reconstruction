# Reconstructing graph diffusion history from a single snapshot

Official code for [the KDD 2023 paper](https://doi.org/10.1145/3580305.3599488).

If you use our code, please cite our paper:

## Dependencies

Our code was tested under the following dependencies:

- CUDA 11.4
- `torch==1.7.0`
- `class-resolver==0.3.10`
- `torch-scatter==2.0.7`
- `torch-sparse==0.6.9`
- `torch-cluster==1.5.9`
- `torch-geometric==2.0.4`
- `ndlib==5.1.1`
- `geopy==2.1.0`
